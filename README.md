# 🚀 TRACKSPARK – Habit Tracker Web App



TRACKSPARK is a simple and modern Habit Tracking Web Application that helps users build productive habits and maintain consistency through streak tracking.

The application allows users to add habits, track their progress, maintain streaks, and stay motivated with visual progress tracking.

# 📌 Project Overview

Developing good habits requires consistency and discipline.
TRACKSPARK helps users maintain their daily habits by tracking progress and encouraging streak continuity.

Users can:

Add daily habits

Track progress

Maintain streaks

Stay motivated

Monitor productivity routines

# ✨ Features

✅ Clean and modern user interface
✅ Add and manage daily habits
✅ Habit streak tracking
✅ Emoji-based habit representation
✅ Motivational quotes for daily inspiration
✅ Simple and lightweight design
✅ Responsive layout

# 🛠️ Tech Stack
Technology	Purpose
HTML5	Structure of the application
CSS3	Styling and layout
JavaScript (Vanilla JS)	Application logic and habit tracking
Google Fonts	Typography
